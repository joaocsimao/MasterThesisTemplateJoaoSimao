# MasterThesisTemplate-JoaoSimao
 
This repo holds all the test templates that were used in my thesis. 
ASAP 2.0 public data is provided, while IAVE data is not, due to it being confidential. If you want to try out these tests with your own data, simply change where the tests pull the data from, they are built in a modular way for easy use. I also included the code I used to generate the figures for my thesis and the long data format for ASAP. A zip file is included with all the grading outputs for ASAP and the R code to evaluate the long data format.
Any questions or further need of explanation please feel free to contact me.
